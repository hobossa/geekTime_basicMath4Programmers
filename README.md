- 007
    - 图：狄克特斯拉算法 Dijkstra， 无负权边加权图找最快路径。（地图导航）
        - Dijkstra 算法的核心思想是，对于某个结点，如果我们已经发现了最优的通路，那么就无需在将来的步骤中，再次考虑这个结点。
        - 如果边的权重都是正值，那么通路上的权重之和是单调递增的，所以其它通路的权重之和一定大于当前的最小权重，也就是即使存在其它通路，其权重也会比当前最小值大。
